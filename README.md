# 📈 raddit

### 레딧이 말하는 주식 시장, 페니주식만 캐치

</div>

---

레딧에서 매일 수천 편의 주식 게시물이 올라옵니다.
그중 실제 가치가 있는 정보는 손에 꼽습니다.

**raddit**은 레딧 서브레딧의 티커 언급량을 실시간으로 집계하여,
$5 미만 페니주식만 선별해 보여주는 대시보드입니다.

개인 투자자가 시장의 관심이 어디로 향하는지 한눈에 파악할 수 있도록 만들었습니다.

---

## ✨ 핵심 기능

| 기능 | 설명 |
|------|------|
| 📊 **언급 순위** | 레딧에서 가장 뜨거운 티커를 실시간 랭킹으로 |
| 🔎 **페니주식 필터** | $5 미만 종목만 자동으로 걸러내어 화면에 표시 |
| 📈 **기술적 분석** | RSI · MACD · 볼린저밴드 · 이동평균을 한국어로 요약 |
| 🕯️ **캔들 차트** | 종목 클릭 한 번으로 가격 흐름 시각화 |
| 📋 **변경 이력** | 서비스 업데이트를 투명하게 공개 |

## 💡 이런 분께 추천합니다

- **개인 투자자** — 아침마다 시장 심리를 빠르게 체크하는 루틴용
- **페니주식 입문자** — 어떤 종목이 주목받는지 트렌드 파악용
- **시장 분석에 관심 있는 분** — 레딧 언급량과 실제 주가의 상관관계 관찰용
